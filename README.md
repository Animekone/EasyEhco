# EasyEhco
A simple script to use Ehco Tunnel
```bash
wget -O ehco.sh https://ghproxy.com/https://github.com/owogo/EasyEhco/blob/main/ehco.sh && bash ehco.sh
```
再次使用脚本： ```bash ehco.sh```
